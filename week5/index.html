Brief Notes

  • Console.log can show us what is really happening.
  • Debuggers are used to create breakpoints.
  • There are several different error objects:
    • Range: outside an available range
    • Reference: reference to an item that does not exist
    • Syntax: code error
    • Type: value type error/wrong data type
    • URI: problem coding/decoding URI
    • Internal: non-standard JS error
  • Try-Catch handles exceptions and returns something when an error is caught.
  • Finally runs regardless of whether there was an error.
