Brief Notes

  • Required means that a field must be entered.
  • Regex can be used to specify what must be entered.
  • JS can be used to validate the form.
  • :valid and :invalid are CSS properties that let us assign styles depending on whether or not the form is valid.
  • Fetch is used to pull in JSON files and other files.
  • Fetch can also be used to upload when used with formData().
  • Request objects can be passed into a fetch.
  • You can pass an old request in a new one to create a copy.
