Week 1 Exercises, Notes, and Questions

Exercises

  • Story Exercise

Notes

  • Playing with code is a good way to learn
  • It is important to take code apart and understand how it works and what each piece does
  • Keys are like labels or titles; they help us find what we are seeking
  • Values are content
  • Use .value with input and textarea
  • Use questions in user content.
  • Use labels and placeholders when creating forms.
  • Display informative error messages next to where they occur.

Questions

  • Why do we not use innerHTML with input and textarea? Why is it .value instead?
  • Why is it a good idea to use questions in our content?
