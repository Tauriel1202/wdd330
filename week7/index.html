Brief Notes

  • Async code runs in the background.
  • Fetch is used to pull in APIs.
  • IIFE is an immediately invoked function. This means that it runs as soon as it is built.
  • Function.length is the number of parameters a function has.
  • This is the first parameter is the argument.
  • Null can be used in place of this when no this is provided.
