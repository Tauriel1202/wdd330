Week 8 Exercises, Notes, and Questions

Exercises

  • Team: Star Wars API
  • Playground: Transforms and Svgs

Brief Notes

  • Transform has different properties that can be used to move or resize HTML.
  • Transitions can be created using only CSS.
  • Scale changes the size of an element, but does not make space for it. Be careful of overlap.
  • Transitions move/change elements over a duration of time.
  • We can use canvas to create a drawing board.
  • Canvas shapes can have img backgrounds.
  • Svgs maintain their quality at any size.
