Brief Notes

  • 'This' is the value before the dot.
  • Arrow functions do not have a 'this', so the value is taken from outside.
  • Parse takes a string and returns a JS Object.
  • Stringify takes a JS Object and returns a JSON file.
  • Object.keys(arrayName) finds all the keys.
  • Object.values(arrayName) finds all the values.
